为什么上进好学的你却始终碌碌无为？

发表于 2017-03-06 | 分类于 Miscellaneous


我们总是认为爱学习的人比爱玩的人更容易取得成功，至少这是在学生阶段的普遍认识。然而，工作以后，随着工作年龄的一点点增长，这句话变得越来越没有说服力，尤其是在金钱至上的单一价值观舆论中，我们甚至于嘲笑说这样话的人。

这实在是一种可悲的现象，究竟这句话哪里错了？为什么我们会觉得学习没用？我相信很多人已经不再思考这个问题，或许从没思考过这个问题，但你必须思考，否则你家的小孩将重走你走过的路。

当我们谈学习时，我们有没有想过我们为什么要学习，我想玩游戏的人很清楚他为什么要玩，为了通关，为了打败对手，他有一个很明确的目标。但是学习呢？我们不知道为什么要学习，更别说怎么学，学什么。

下面我就从几个现象来分析为什么你一直在学习却始终碌碌无为：

1. 以为自己在学习

这一类人其实有点可悲，因为他以为自己在学习，搞得很忙，其实只是在消遣自己的时间。哪类人呢？整天泡在知乎、果壳、维基百科的人。这些人，上午在研究美国三权分立制度，下午在研究西方美术史，晚上还得分析南海问题和《开罗宣言》。别人问起他，你业余时间都干嘛呢，他回答不上来。跟人聊天的时候觉得自己什么都懂，但就是讲不出来，讲不清楚。写年终总结的时候会感觉没什么东西可写，然后自我安慰，我这是在积累，总有一天会爆发。嘿，该醒醒了，你看的那些东西只是打发时间，根本算不上学习。

2. 从入门到放弃

我相信大家对这句话已经不陌生了，谁没有几段从入门到放弃的经历呢？这并不是什么坏事，但如果你的学习经历中全是从入门到放弃，那就有问题了。学一门计算机语言，花两天配好开发环境，花100块买书，然后就没有然后了；学日语，还没背完50音图，就已经忘记自己为什么而开始；练力量，只去了两节课；学吉他，手无茧，弦已锈。喂~，请尊重下自己的时间好不好，说实在，现在的各种诱惑太多了，在我们开始一段学习之前，花20分钟认真想想，我真的喜欢么？我的目的是什么？我能坚持多久？如果想不明白，那就把它记在一本专门记录兴趣冲动的本子上，过一周再去看，如果还是非常想，那就开始。我保证这个本子上的兴趣会越来越多，而真正实施的几乎没有。

3. 好学习，不求甚解

陶渊明说：“好读书，不求甚解”，这是另外一种状态，闲适自在，完全享受读书的过程，我们当然也追求这样一种状态，但是今天，我们的追求不一样，学习更多时候是功利性的，带有明确的目的。很多人会在各种网络公开课中追课，幸福学、心理学、逻辑学。。。一门不拉下，学的过程也时常能有心灵碰撞的机会，偶感欢欣。这些人是幸福的，但还是我们开始讲的，这样的学习并不会给我们的生活事业带来多大的成就。首先还是我们的目标不明确，其次是学习没有压力，学多少算多少，但是时间是有限的，我们应该在这有限的时间中让学习的效果最大化。如果是带有互动的课程，应该先做好预期，带着问题去听课，争取在课中消化所有知识点。然后是总结，把所有听到的学到的知识点与自己的原有认知结合，举一反三，学以致用，这样才算是真正学到了这个知识点。

4. 不成体系

如果你已经认认真真学完了某一门课程，并且充分融入自己的生活工作中，那肯定会有看得见的帮助。比如，你花两个月学完手机摄影课程，肯定不会再被女朋友嫌弃了，朋友圈的点赞数也会有可观的增长。你当然可以就此停止，但如果你想有更大的突破，恭喜你，当你有这样的想法时，你已经超过了99%的人。接下来你应该去接触真正摄影界的大牛，体系化学习，学构图、学明暗光影还只是技术，更多的时候，你在学如何表达世界，洞悉内心。

5. 无功利，不学习

我不知道有多少人能真正说出这句话，“XXX就是我的生命”，能说出来的都是上帝的宠儿，他们的成就更多的是被兴趣驱动。遗憾的是，上帝没有那么多宠儿，当我们学某项技能到达一个瓶颈期后就会停滞不前，这时候我们也许会迷茫，有时候会放弃，这种情况多发生在学生阶段。学了5年的吉他，说放弃就放弃了；学了4年的哲学，毕业选择去做销售。想想真是蛮可惜的，说不爱，也许是欺骗自己，说爱，却难以坚持。哎，还是谈钱吧，这确确实实是一条坚持下去的理由，或者叫商业化，当你觉得不想坚持的时候，想想怎么把它变现，听我的，金钱回报会刺激你坚持把它做好，名利双收，不是更好吗？

亲爱的，也许你并不认同以上观点，没关系，每个都有自己的生活方式。不过，当你为终日苦苦学习而不到应有的回报，自暴自弃时，看看是不是对号入座了。
